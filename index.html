元となるデータをとってきて加工する

　参院選の得票を記録した朝日新聞の社内データベースと、旧統一教会がウェブサイトで公表している「家庭教会」という教団施設の住所を元に作ったCSVファイル「UC_inoue.csv」を読み込む。教団施設の有無は「UC有無」とタグ付けしている。「HS有無」「HC有無」はそれぞれ「幸福の科学」の施設と保健所の有無を意味しているが、今回の分析には使っていない。その他のデータは政府の「社会・人口統計体系」をeStatから取得して加工したもの。

inoue <- read_csv("data/UC_inoue.csv") %>%
  mutate(inoue_ratio=井上得票*100/投票者数, # 井上氏の得票率と
         ldp_ratio=(自民得票計-井上得票)*100/投票者数) %>% # 井上氏を除いた自民の得票率を算出

  pivot_wider(names_from = 年,
              values_from = c(inoue_ratio,
                              ldp_ratio)) %>%
  mutate(inoue_ratio_change=inoue_ratio_2022-inoue_ratio_2019, # 井上氏の得票率変化量と

  dplyr::select(-inoue_ratio_2022,
                -inoue_ratio_2019,
                -ldp_ratio_2022,
                -ldp_ratio_2019) %>% 
  dplyr::select(都道府県市区町村,
                inoue_ratio_change,
                UC有無,
                HS有無,
                HC有無,
                ldp_ratio_change,
                `A2301_住民基本台帳人口（総数）`,

マッチングをしていく

　以下の３モデルのそれぞれについて、最近傍マッチングと遺伝的マッチングを行う。つまり計６パターンのマッチングをする。

  1. 社会・人口統計体系から入手した67個の変数すべてに、井上氏以外の自民党得票率を加えた計68個
  2. 社会・人口統計体系の変数全体のうち効果量の大きいもの10個に、井上氏以外の自民党得票率を加えた計11個
  3. 社会・人口統計体系の各分野のうち効果量が最大の変数8個に、人口と井上氏以外の自民党得票率を加えた計10個

　次に遺伝的マッチングをする。コードはチャンクの中に埋め込んでいるが、チャンクの中身を表示すると、膨大な計算の途中経過まで出力されてしまうため、やむを得ず非表示にしてある。マッチングのための様々なパラメータは以下の通り。

method = “genetic”,

distance = “mahalanobis”,

pop.size = 1000,

nboots = 1000,

replace = TRUE,

verbose = TRUE,

estimand = “ATT”
